Modules Configuration
=====================

Les modules de configuration fournissent une architecture modulaire et flexible pour gérer tous les paramètres de l'application. Chaque module est spécialisé dans un aspect de la configuration, comme la base de données, les modèles de langage ou l'envoi d'e-mails, et hérite des fonctionnalités de ``BaseConfig``.

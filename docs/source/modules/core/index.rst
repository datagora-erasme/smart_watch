Modules Core
============

Les modules Core constituent le cœur de l'application, orchestrant les fonctionnalités essentielles telles que la gestion de la configuration, la communication avec les modèles de langage (LLM), la gestion de la base de données, et le traitement des erreurs. Ils forment la base sur laquelle les autres modules de l'application s'appuient.
